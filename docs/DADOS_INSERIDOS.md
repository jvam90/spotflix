## 🔍 CONSULTAS ÚTEIS PARA VERIFICAR OS DADOS

### Verificar Usuários
```sql
SELECT * FROM "AspNetUsers" WHERE "Email" LIKE '%@spotflix.com';
```

### Verificar Planos
```sql
SELECT * FROM "Plans";
```

### Verificar Bandas e Álbuns
```sql
SELECT b."Id", b."Name", b."Genre", b."FormedYear", 
       COUNT(a."Id") as album_count
FROM "Bands" b
LEFT JOIN "Albums" a ON a."BandId" = b."Id"
GROUP BY b."Id", b."Name", b."Genre", b."FormedYear";
```

### Verificar Músicas por Álbum
```sql
SELECT a."Title", COUNT(s."Id") as song_count
FROM "Albums" a
LEFT JOIN "Songs" s ON s."AlbumId" = a."Id"
GROUP BY a."Id", a."Title";
```

### Verificar Cartões por Usuário
```sql
SELECT u."FullName", c."Brand", c."Last4", c."AvailableLimit", c."Active"
FROM "AspNetUsers" u
LEFT JOIN "Cards" c ON c."UserId" = u."Id"
ORDER BY u."FullName";
```

### Verificar Transações
```sql
SELECT c."HolderName", t."Merchant", t."Amount", t."OccurredAt", t."Status"
FROM "Transactions" t
JOIN "Cards" c ON c."Id" = t."CardId"
ORDER BY t."OccurredAt" DESC;
```

### Verificar Assinaturas
```sql
SELECT u."FullName", p."Name", s."Status", s."StartedAt", s."CurrentPeriodEnd"
FROM "Subscriptions" s
JOIN "AspNetUsers" u ON u."Id" = s."UserId"
JOIN "Plans" p ON p."Id" = s."PlanId";
```

### Verificar Favoritos de Músicas
```sql
SELECT u."FullName", s."Title", a."Title" as album
FROM "FavoriteSongs" fs
JOIN "AspNetUsers" u ON u."Id" = fs."UserId"
JOIN "Songs" s ON s."Id" = fs."SongId"
JOIN "Albums" a ON a."Id" = s."AlbumId"
ORDER BY u."FullName";
```

### Verificar Favoritos de Bandas
```sql
SELECT u."FullName", b."Name"
FROM "FavoriteBands" fb
JOIN "AspNetUsers" u ON u."Id" = fb."UserId"
JOIN "Bands" b ON b."Id" = fb."BandId"
ORDER BY u."FullName";
```
